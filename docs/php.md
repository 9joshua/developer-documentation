# PHP

## Xdebug

### Recommended settings for Xdebug 3 and newer

```ini
[xdebug]
xdebug.mode=debug
xdebug.start_with_request=yes
xdebug.log_level = 0
```

This can be useful for also debugging php invoked from the CLI.
